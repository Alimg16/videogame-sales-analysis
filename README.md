# videogame-sales-analysis
Análisis exploratorio de datos de ventas de videojuegos para identificar tendencias de mercado, rendimiento de la plataforma y factores asociados al éxito comercial.

Descripción
Este proyecto analiza información histórica de ventas de videojuegos para identificar los factores que influyen en el éxito comercial de un título. A partir del análisis de plataformas, géneros, reseñas de críticos y usuarios, clasificaciones ESRB y preferencias regionales, se generan conclusiones que ayudan a orientar estrategias de marketing y la planificación de campañas para futuros lanzamientos.

Objetivo
Explicar qué factores influyen en el éxito comercial de un videojuego y generar recomendaciones para campañas de marketing en 2017.

Dataset
- games.csv
- Variables principales:
  - Plataforma
  - Género
  - Año de lanzamiento
  - Ventas por región
  - Calificaciones de críticos y usuarios
  - Clasificación ESRB

Tecnologías utilizadas
- Python
- Pandas
- NumPy
- Matplotlib
- SciPy

Principales resultados
- PS4 y XOne fueron las plataformas con mayor potencial hacia 2017.
- Shooter presentó la mayor rentabilidad promedio por juego.
- Las reseñas de críticos mostraron una correlación moderada con las ventas, mientras que las de usuarios prácticamente no presentaron relación.
- Norteamérica y Europa mostraron preferencias similares, mientras que Japón presentó un comportamiento de mercado distinto.
- Las pruebas de hipótesis mostraron que las calificaciones entre Xbox One y PC no difieren significativamente, mientras que los géneros Action y Sports sí presentan diferencias.
